## Instaliation

1. Clone this repository:
```
$ git clone https://github.com/FrozenPyrozen/blog-task.git
```
2. Change the current directory to **blog-task**:
```
$ cd ./blog-task
```
3. Install dependencies using npm/yarn:
```
$ npm install
```
```
$ yarn install
```
4. Run app in terminal:
```
$ npm start
```
